The modpack is made with balance in mind, while still providing objectives in progressing some of the main mods.
Mods which ruin the flow of the game, or overpower the player have been left out.

Main Mod List:

  • A Man With Plushies : Adds a gacha-like mechanic used to obtain plushies from a variety of pop-culture media.
  • Additional Lanterns : Adds lantern variety and functionality.
  • Animal Feeding Trough : Adds a block which lets you automatically feed and breed animals.
  • Artifacts : Adds mimics, which drop artifacts upon defeat. These can give small buffs.
  • Automobility : Adds cars. These are shit for traveling but fun for racing :)
  • Basket : Basket.
  • Beautify Refabricated : Adds vanilla style decor.
  • Blahaj : Blahaj.

  • Comforts : Adds sleeping bags and hammocks.
  • Convenient Decor : Adds decorative blocks, some with convenient function.
  • Create : Adds a mechanics system, where blocks powered by rotation interact with vanilla mechanics.
  • Create Steam 'n Rails : Adds trains :3
  • Create Estrogen : Estrogen.
  • Creeper / Enderman Overhaul : Adds biome-specific variants of the afforementioned mobs.
  • Dark Painting : More painting variation.
  • Diagonal Fences : Only works on Macaw's fences, less cursed than it sounds.
  • Extra Origins : Adds 4 more origins to the origins mod.
  • End Remastered : Changes the buildup to the end portal. The eyes are found by exploring, fighting, and mining.
  • Exline's Furniture : Froggy chair :3 (and other furniture)
  • Expanded Storage : Adds more and advanced storage options.
  • Friends&Foes : Adds all the unpicked mob vote contestants.
  • Handcrafted : Adds furniture in multiple different styles.
  • Incendium : Expands on the Nether. New biomes, mobs, items, and advancements.
  • Let's Do Vinery : Adds a crafting system for creating wine, including new bushes, trees, and some decorative blocks.
  • Lootr : Loot chests have player specific loot, no more running into empty and looted structures :3
  • Macaw's mods : Adds stylised bridges, doors, fences, walls, paths, pavings, trapdoors and windows.
  • More Axolotl : Adds more axolotl colours, which can breed to create combined colours.
  • Nature's compass : Compass to find biomes uwu
  • Night Lights : Adds more lighting options.
  • Omni-hopped : Adds omni-directional hoppers.
  • Origins : Adds Origins to the game, which are conditions you play with that are both positive and negative at times.
  • Practical Plushies : Adds Minecraft-mob themed plushies.
  • Respawnable Pets : Adds an item which lets your pets respawn.
  • Terralith : Adds more biome variation.
  • Touhou Origins : Adds more variation in origin choice :3
